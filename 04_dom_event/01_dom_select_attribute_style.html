3. HTML Attribute와 DOM Property

HTML Attribute는 HTML에 작성된 초기값에 가깝고, DOM Property는 JavaScript가 현재 다루는 최신 상태에 가깝다.

4. data attribute와 dataset

data-* 속성은 HTML 요소에 사용자 정의 데이터를 저장할 때 사용한다. JavaScript에서는 dataset으로 접근한다.
