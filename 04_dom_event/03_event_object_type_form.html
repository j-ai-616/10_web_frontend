03_이벤트 객체와 폼 이벤트

이벤트 객체에는 이벤트가 발생한 요소, 입력값, 키보드 정보 같은 세부 정보가 담긴다.

1. 이벤트 객체

이벤트가 발생하면 브라우저는 이벤트 정보를 담은 객체를 만들고, 이벤트 핸들러의 첫 번째 매개변수로 전달한다.

버튼을 클릭해보세요.

2. 마우스 이벤트

마우스 클릭, 더블 클릭, 마우스 진입/이탈 등은 모두 이벤트로 처리할 수 있다.

마우스를 올려보세요.

3. 키보드 이벤트

키보드 입력은 keydown, keyup 이벤트로 확인할 수 있다. 입력된 실제 문자는 event.key로 확인한다.

입력한 키 정보가 여기에 표시된다.

4. 사용자 입력 양식 이벤트

폼 요소는 값을 입력하거나, 선택하거나, 포커스를 얻고 잃을 때 다양한 이벤트가 발생한다.




0/100자

5. 폼 제출 이벤트

submit 이벤트는 폼을 서버로 전송하기 전에 입력값을 검증할 때 자주 사용한다.

검색어를 입력한 뒤 검색 버튼을 눌러보세요.
