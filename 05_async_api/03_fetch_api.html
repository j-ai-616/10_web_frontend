03_fetch API 요청

fetch()는 브라우저에서 기본으로 제공하는 HTTP 요청 함수이다. 별도 라이브러리를 추가하지 않아도 사용할 수 있으며, 요청 결과를 Promise로 반환한다.

1. fetch와 async/await

서버에 요청을 보내면 응답이 바로 도착하지 않을 수 있다. 그래서 fetch()는 Promise를 반환하고, await를 사용해 응답이 도착할 때까지 기다릴 수 있다.

fetch()의 응답 객체는 HTTP 응답 전체를 의미한다. 실제 JSON 데이터는 response.json()을 호출해서 한 번 더 변환해야 한다.

버튼을 클릭하면 API 요청 결과가 표시된다.
