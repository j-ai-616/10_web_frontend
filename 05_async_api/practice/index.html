비동기 API 종합 실습 - 게시글과 댓글 조회

fetch와 axios를 사용해 외부 API에 데이터를 요청하고, 응답 데이터를 화면에 출력한다.

실습 안내

게시글 목록 조회는 fetch로 구현하고, 댓글 조회와 게시글 작성 요청은 axios로 구현한다.

JSONPlaceholder는 연습용 API이므로 POST 요청을 보내도 실제 데이터가 저장되지는 않는다. 다만 서버가 성공 응답을 반환하므로 API 요청과 응답 처리 흐름을 연습할 수 있다.

버튼을 클릭해 게시글 목록을 불러오세요.
