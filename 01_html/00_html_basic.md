# 00. HTML 기본 개요

## 1. HTML이란?

HTML은 웹 페이지의 구조와 내용을 작성하는 표준 마크업 언어이다.

웹 브라우저는 HTML 문서를 읽고 해석하여 사용자에게 화면으로 보여준다.  
HTML은 제목, 문단, 목록, 표, 이미지, 링크, 입력 양식처럼 화면에 필요한 내용을 태그로 표현한다.

HTML은 프로그래밍 언어라기보다 문서의 구조를 표시하는 언어이다.  
따라서 HTML 수업에서는 "무엇을 화면에 배치할 것인가"와 "각 내용이 어떤 의미를 가지는가"를 중심으로 학습한다.

## 2. 태그, 요소, 속성

HTML 문서는 태그를 사용하여 작성한다.

```html
<p class="description">HTML 기본 문단이다.</p>
```

위 코드에서 각 구성요소는 다음과 같다.

1. 태그
   - `<p>`, `</p>`처럼 꺾쇠괄호로 감싼 명령어이다.
   - 시작 태그와 종료 태그로 내용을 감싼다.

2. 요소
   - 시작 태그, 내용, 종료 태그를 포함한 전체 구조이다.
   - `<p class="description">HTML 기본 문단이다.</p>` 전체가 하나의 요소이다.

3. 속성
   - 태그에 추가 정보를 부여한다.
   - `class="description"`처럼 속성명과 속성값으로 작성한다.

## 3. HTML 기본 구조

HTML 문서의 기본 구조는 다음과 같다.

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>문서 제목</title>
</head>
<body>
    <h1>화면에 보이는 제목</h1>
    <p>화면에 보이는 문단이다.</p>
</body>
</html>
```

각 영역의 역할은 다음과 같다.

1. `<!DOCTYPE html>`
   - 현재 문서가 HTML5 문서임을 선언한다.

2. `<html>`
   - HTML 문서 전체를 감싸는 최상위 요소이다.

3. `<head>`
   - 문서 제목, 문자 인코딩, CSS 연결 정보처럼 화면에 직접 보이지 않는 문서 정보를 작성한다.

4. `<body>`
   - 실제 브라우저 화면에 표시되는 내용을 작성한다.

## 4. head 영역에서 자주 사용하는 태그

`<head>` 영역에는 브라우저 화면에 직접 출력되는 내용보다 문서 정보를 설명하거나 외부 파일을 연결하는 태그를 작성한다. `meta charset`은 문서의 문자 인코딩을 지정하여 한글이 깨지지 않도록 도와주고, `viewport`는 모바일과 같은 다양한 화면 크기에서 페이지가 적절하게 보이도록 기준을 설정한다. `title`은 브라우저 탭에 표시되는 문서 제목을 지정한다. `link`는 외부 CSS 파일을 연결할 때 주로 사용하며, `script`는 JavaScript 파일을 연결하거나 문서 안에 JavaScript 코드를 작성할 때 사용한다.

## 5. HTML 작성 시 주의사항

HTML을 작성할 때는 다음 사항을 지키는 것이 좋다.

1. 태그명은 소문자로 작성한다.
2. 시작 태그를 작성했다면 가능한 종료 태그도 함께 작성한다.
3. 파일 확장자는 `.html`로 저장한다.
4. 들여쓰기를 사용하여 부모 요소와 자식 요소의 관계를 보기 쉽게 작성한다.
5. 단순히 보이는 모양만 생각하지 말고, 태그가 가진 의미를 고려한다.
