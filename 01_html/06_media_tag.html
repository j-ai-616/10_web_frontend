미디어 관련 태그

오디오 관련 태그

audio 태그는 웹 문서에 오디오 파일을 삽입할 때 사용한다.
controls 속성을 사용하면 재생, 정지, 음량 조절 등의 기본 컨트롤을 표시할 수 있다.

source 태그를 사용한 오디오 파일 지정

source 태그를 사용하면 하나의 audio 태그 안에 여러 형식의 파일을 지정할 수 있다.
브라우저는 지원 가능한 형식의 파일을 선택하여 재생한다.

비디오 관련 태그

video 태그는 웹 문서에 비디오 파일을 삽입할 때 사용한다.
controls 속성을 사용하면 사용자가 직접 영상을 재생하거나 일시 정지할 수 있다.

비디오 주요 속성

  • controls : 재생 컨트롤을 표시한다.
  • autoplay : 페이지가 열리면 자동 재생을 시도한다.
  • muted : 음소거 상태로 재생한다.
  • loop : 재생이 끝나면 반복 재생한다.
  • preload : 페이지 로드 시 미디어 파일을 어느 정도 미리 불러올지 지정한다.

최근 브라우저는 사용자 경험 보호를 위해 소리가 있는 자동 재생을 제한하는 경우가 많다.
따라서 autoplay를 사용할 때는 muted를 함께 사용하는 경우가 많다.
