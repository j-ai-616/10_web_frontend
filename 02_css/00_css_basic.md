# 00. CSS 기본 개요

## 1. CSS란?

CSS는 HTML 문서에 디자인과 레이아웃을 적용하는 스타일시트 언어이다.

HTML이 웹 페이지의 구조와 내용을 담당한다면, CSS는 화면의 글꼴, 색상, 여백, 배경, 크기, 배치 등을 담당한다.

HTML과 CSS를 분리하면 다음과 같은 장점이 있다.

1. 문서 구조와 디자인을 구분할 수 있다.
2. 같은 HTML 구조에 다른 디자인을 적용할 수 있다.
3. 여러 페이지의 스타일을 하나의 CSS 파일에서 관리할 수 있다.
4. 유지보수와 수정이 쉬워진다.

## 2. CSS 기본 문법

CSS는 선택자와 선언 블록으로 작성한다.

```css
p {
    color: blue;
    font-size: 16px;
}
```

각 구성요소는 다음과 같다.

1. 선택자
   - 스타일을 적용할 HTML 요소를 지정한다.
   - 위 예시에서는 `p` 태그가 선택자이다.

2. 속성
   - 바꾸고 싶은 스타일 항목이다.
   - `color`, `font-size`가 속성이다.

3. 값
   - 속성에 적용할 구체적인 설정이다.
   - `blue`, `16px`가 값이다.

## 3. CSS 적용 방식

CSS를 HTML에 적용하는 방식은 크게 세 가지이다.

## 3-1. 인라인 스타일

HTML 태그의 `style` 속성에 직접 CSS를 작성하는 방식이다.

```html
<p style="color: red;">빨간색 문단이다.</p>
```

간단히 확인할 때는 사용할 수 있지만, 스타일이 HTML 안에 섞이므로 수업에서는 주된 방식으로 사용하지 않는다.

## 3-2. 내부 스타일 시트

HTML 문서의 `<head>` 영역에 `<style>` 태그를 작성하는 방식이다.

```html
<head>
    <style>
        p {
            color: blue;
        }
    </style>
</head>
```

하나의 HTML 파일 안에서 예제를 빠르게 확인할 때 유용하다.

## 3-3. 외부 스타일 시트

CSS 파일을 따로 만들고 HTML 문서에서 연결하는 방식이다.

```html
<link rel="stylesheet" href="style.css">
```

실제 웹 페이지를 만들 때 가장 많이 사용하는 방식이다.  
HTML은 구조를 담당하고 CSS는 디자인을 담당하도록 역할을 분리할 수 있다.

## 4. 선택자와 캐스케이딩

CSS는 선택자를 사용하여 특정 요소에 스타일을 적용한다.

대표적인 선택자는 다음과 같다.

1. 태그 선택자
   - `p`, `h1`, `div`처럼 태그 이름으로 선택한다.

2. 클래스 선택자
   - `.menu`, `.card`처럼 여러 요소에 공통 스타일을 적용할 때 사용한다.

3. 아이디 선택자
   - `#header`, `#title`처럼 문서에서 하나의 요소를 식별할 때 사용한다.

CSS는 여러 스타일이 동시에 적용될 수 있다.  
이때 어떤 스타일이 최종적으로 적용되는지는 우선순위, 작성 순서, 상속 여부에 따라 결정된다.  
이 흐름을 캐스케이딩이라고 한다.
